Tema 4: Análisis de frecuencia de palabras y patrones lingüísticos en textos.

Frecuencia

Frecuencia

frase="python es divertido y poderoso"

frase_limpio=frase.replace(",","").replace(",","").lower()

tokens=frase_limpio.split()

frecuencia={}
for palabra in tokens:
    if palabra in frecuencia:
        frecuencia[palabra] += 1
    else:
        frecuencia[palabra] = 1

print(frase_limpio)

📚 Glosario de Funciones

if

se utiliza para ejecutar un bloque de código solo si se cumple una condición determinada. Es una estructura de control muy común en la programación.

else

La cláusula else se usa junto con if (y opcionalmente elif) para ejecutar un bloque de código cuando ninguna condición anterior se cumple.

split

La función split() en Python se usa para dividir una cadena de texto en partes (normalmente palabras), y es clave para generar tokens de manera sencilla.

lower

La función lower() en Python se utiliza para convertir todo un texto o un token (palabra) a minúsculas.
